# 🚀 一键部署 - 快速指南

## ✅ 所有配置已完成，你只需要：

---

## 方式 1：双击运行部署脚本（最简单）

```
双击：deploy.bat
```

然后：
1. 按任意键 → 自动打开浏览器登录
2. 用 **GitHub 账号** 登录
3. 选择 `Create new project`
4. 等待部署完成

**完成！** 🎉

---

## 方式 2：手动命令（更灵活）

### 步骤 1：打开浏览器创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名：`quant-platform`
3. 点击 `Create repository`

### 步骤 2：推送代码到 GitHub

```bash
cd C:\Users\huangruiran\ai_test
git remote add origin https://github.com/你的用户名/quant-platform.git
git push -u origin main
```

### 步骤 3：运行部署

```bash
railway login
railway init
railway up
```

### 步骤 4：绑定域名

```bash
railway domain
```

---

## 🔐 登录时会打开浏览器

选择：
- ✅ **Login with GitHub**
- 授权 Railway 访问你的仓库

---

## 📊 部署后会得到

```
https://quant-platform-xxx.railway.app
```

---

## ⚠️ 重要：设置环境变量

部署完成后，在 Railway 控制台添加：

| 变量名 | 值 |
|--------|-----|
| `AZURE_OPENAI_KEY` | 你的 Azure 密钥 |
| `AZURE_OPENAI_ENDPOINT` | 你的 Azure 端点 |
| `GOOGLE_APPLICATION_CREDENTIALS` | 你的 GCP 凭证 |

---

## 🆘 遇到问题？

```bash
# 查看部署状态
railway status

# 查看日志
railway logs

# 重新部署
railway up --force
```
